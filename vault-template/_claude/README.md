# AI-generated content

Files in this folder were created by Claude Code and are awaiting your review.

## What to do with these files

**To keep a note:** move it to wherever it belongs in your vault (e.g. `notes/`, `Projects/MyProject/notes/`).

**To discard:** delete the file. Nothing in your main vault will break.

**To mark as reviewed (without moving):** change `claude_status: draft` to `claude_status: reviewed` in the frontmatter. Claude will treat it as acknowledged.

## Why staging?

Claude Code creates notes here rather than directly in your vault so you always have a clear picture of what's human-created vs AI-created. Your main note tree stays yours.

Notes in `_claude/` folders do not appear in your Obsidian graph until you move them.

## Status values

| Frontmatter value | Meaning |
| --- | --- |
| `claude_status: draft` | Created by Claude, not yet reviewed |
| `claude_status: reviewed` | You've seen it, chose to leave it here |
| `claude_status: approved` | Approved but not moved yet |

Run `/review-generated` in Claude Code to see a full list of pending notes across your vault.
